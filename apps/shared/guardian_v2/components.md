# Guardian V2 共用元件簡述

- **GuardianHeader**：呈現 LOGO、主選單 pills 與 A/B 模式切換 segmented control。
- **GuardianHeroCard**：顯示核心 KPI、方案資訊與主要 CTA（查看完整報表 / 下載 PDF 或建立警報）。
- **GuardianSearchFilters**：城市 / 店名 Autocomplete 欄位，採 input + datalist 基礎互動。
- **GuardianInsightCard**：安置洞察類型標籤、重點摘要、CTA；依 severity 改變邊框與配色。
- **GuardianAlertList**：列出競品事件或守護任務，包含時間、摘要與建議行動按鈕。
